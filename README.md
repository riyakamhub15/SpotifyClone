# 🎧 Spotify Clone (HTML, CSS & JavaScript)

# Project Overview
This project is a front-end clone of the Spotify web player built using HTML, CSS, and JavaScript. It replicates the look and feel of Spotify while implementing basic music player functionality such as play, pause, and track control.

---

# Features
-  Play and pause songs  
-  Next and previous track controls  
-  Progress bar (seek functionality)  
-  Volume control  
-  Clean and modern Spotify-inspired UI  
-  Responsive design for different screen sizes  

---

# Tech Stack
- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout design  
- **JavaScript (Vanilla JS)** – Functionality and interactivity

# How to Run the Project
1. Clone the repository:
   ```bash
   git clone https://github.com/riyakamhub15/SpotifyClone.git
   ```
2. Open the project folder  
3. Run `index.html` in your browser

# Limitations
- This is a **front-end only project**  
- No backend or database integration  
- Limited song collection (local files only)
